# Daily Dose Of Memes

## What is it?

-Well essentailly its a web-based meme geenerator that pulls memes from reddit using meme API and well displays them, Nothing complicated just a simple website

## How does it work?

-Uses classic Html for the base , css for the graphics and JS for the workings
-The .js file fetches about 25 memes per session and just cycles through it using the button 

## How to run
-Just open index.html in a browser or visit the live site.


## Tech Stack
- HTML
- CSS  
- JavaScript (Vanilla)
